* Introduction
  - Beta denudation, reduction of the beta phase, seen in some surface
    samples of Ti-64.
  - This microstructural degredation increases potential for failure.
  - Detection usually a manual process, involving measurement by hand,
    with human judgement, one-by-one.
  - Automation of detection will provide a quick and useful "second
    opinion" to guide judgement of whether a sample is beta
    denudated.
* Motivation
* Method
  - Process optical image to enhance the alpha-beta microstructure.
  - Extract the alpha-beta volume fraction from image.
  - Analyse volume fraction data to give a predictor variable
  - Feed the predictor variable into the machine learning model which
    will give a "yes/no" answer as to whether the sample is denudated,
    with a probability.
  - Repeat for all images.

** Technical details
   - Train a machine learning model (support vector machine/logistic
     regression) on a training set of data.
   - Training set is 200 images, which were classified by myself and
     Chris Collins.
   - Test set of data used to test the predictions of the model on
     unseen data.
   - Accuracy score gives our confidence in the machine learning
     model.
